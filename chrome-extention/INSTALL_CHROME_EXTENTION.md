## Install SmartQR

1. To install SmartQR, First go to the [SmartQR Github Project](https://github.com/divyaxavier/SmartQR/). Download the project as a ZIP folder by clicking the `code` button and then `Download Zip` button. 
<img src="https://raw.githubusercontent.com/divyaxavier/SmartQR/main/web/build/assets/steps-to-install/step1.png" width="70%" >

2. Extract the folder into your desired location
<img src="https://raw.githubusercontent.com/divyaxavier/SmartQR/main/web/build/assets/steps-to-install/step2.png" width="70%" >

3. Now, to install the chrome extention go to [chrome://extensions](chrome://extensions) and click `Load Unpacked` button. Make sure that `Developer Mode` is on.
<img src="https://raw.githubusercontent.com/divyaxavier/SmartQR/main/web/build/assets/steps-to-install/step3.png" width="70%" >

4. Select the `extention` folder from your SmartQR folder and click `Open` button.
<img src="https://raw.githubusercontent.com/divyaxavier/SmartQR/main/web/build/assets/steps-to-install/step4.png" width="70%" >

5. Now you could see a new extention at your [chrome://extensions](chrome://extensions) dashboard
<img src="https://raw.githubusercontent.com/divyaxavier/SmartQR/main/web/build/assets/steps-to-install/step5.png" width="70%" >
